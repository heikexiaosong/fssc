#### 基本利率信息
- /sys/fms/fmsBaseInterest/page
```json
{
  "query": [{"code":"interestType","condition":14,"fieldType":1,"value":"01"}]
}
```

- interestType: 利率类型； 01：1年期LPR； 06：5年期LPR
- finacingType：融资类型； 1：贷款； 
- financeCategory: 融资类别； 2：融资；


#### 利率自动调整
- /sys/fms/fmsLoanInterestRecord/autoAdjustLoan
```json
{
  "provideLoanId":"24f66eed2704c61d018a3139322e6e30"}
```

```sql

-- 根据贷款id+调整日期 查询lpr调整日期（仅查询今日及以前）
select * 
  from FMS_PROVIDE_LPR_DATE
 where PROVIDE_ID = :id
   and enabled_flag = '0'
   and LPR_ADJUST_DATE <= new Date()

-- 获取贷款的利率种类（以放款上选择的基准LPR为初始值）
-- baseRateEntity
select *
  from FMS_BASE_INTEREST
 where id = loan::baseRateId

-- 组装基准利率key：利率种类、融资品种、币种、日利率天数、状态（有效）； 其中状态，在查询的时候过滤， 不需要在key中使用了
-- key： <baseRateEntity::interestType>_<headerDTO.currencyCode>_<loanDTO::Cardinaliy>
   
```
