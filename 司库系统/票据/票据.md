# 票据

### 1. 收票登记（登记）
* **业务概念**：企业收到客户支付的纸质或电子汇票时，将其录入财务或司库系统的过程。
* **主要场景**：销售回款。客户用票据来抵扣其应收账款。
* **系统处理**：
  * 录入/同步票据核心信息：汇票编码（30位电票编码）、出票人、收票人、承兑人、出票金额、到期日期等。
  * **系统状态**：票据录入成功后，在系统的初始状态通常为 **“已收票”** 或 **“在库”**，可用金额等于出票金额。

### 2. 票据背书（(Bill Endorsement)）
* **业务概念**：持票人在票据背面签字或盖章（电票则在网银或司库系统发起背书申请），将票据权利转让给另一个收款人的行为。
* **主要场景**：对外付款。企业为了减少现金占用，将手里尚未到期的汇票直接转让给自己的供应商，用以抵付货款。
* **系统处理**：
  * 记录“被背书人”（下一个持票人）的信息。
  * 若系统支持**“拆包背书”**，可将大额票据拆分成小额分别转让。
  * **系统状态**：背书成功并被对方签收后，当前企业的可用余额清零，状态变更为 **“已背书”**。

### 3. 票据质押（质押）与质押解除
* **业务概念**：企业将未到期的票据作为担保物交付给债权人（通常是银行），以获取贷款或开立新票据的行为。
* **主要场景**：融资担保。企业急需现金流或需要开立新票，将持有的高信用等级票据质押给银行。
* **系统处理**：
  * **质押**：向银行发起质押申请，审批通过后，票据状态转为 **“已质押”**。在质押期间，票据被冻结，无法发起背书、贴现或托收。
  * **质押解除**：当企业履行了相应债务（如还清贷款）后，银行将票据权利退还。系统状态恢复为 **“已收票/在库”**，重新恢复流通能力。

### 4. 票据贴现（贴现）
* **业务概念**：持票人在票据**到期日前**，将票据转让给银行，银行扣除一定的**贴现利息**后，将剩余资金付给企业的行为。
* **主要场景**：提前变现。企业急需现金流，自愿承担一部分利息损失，以换取即期流动资金。
* **系统处理**：
  * 系统需计算并录入：贴现利率、贴现利息、实际到账金额。
  * **系统状态**：贴现完成后，资金进入企业银行账户，票据所有权转移给银行，系统状态变更为 **“已贴现”**。

### 5. 票据托收（托收）与托收到账
* **业务概念**：票据**到期时**，持票人向承兑银行（或通过自己的开户行）提示付款，要求承兑人无条件支付票款的行为。
* **主要场景**：正常到期兑现。票据即将到期或已到期，企业发起托收，静待资金全额落袋。
* **系统处理**：
  * **托收申请**：在系统中勾选即将到期的票据，发起提示付款申请。系统状态变为 **“托收中”**。
  * **托收到账**：承兑行扣款成功并划转至企业账户后，系统确认生成“托收到账单”，票据生命周期正式终结，状态变为 **“已结清”**。


## 票据流转生命周期图解
[客户付款] ──>  1. 收票登记 (票据入库)
                            │
    ┌───────────────────────┼────────────────────────┐
    ▼                      ▼                        ▼
 2.票据背书             4. 票据贴现               3. 票据质押
(转让给供应商)          (未到期卖给银行)           (作为担保物冻结)
    │                       │                       │
   ▼                       ▼                       ▼
 [退出企业流转]          [提前变现结清]            [质押解除恢复]
                                                    │
                                                    ▼
                                                5. 票据托收
                                                (到期全额拿钱)

## 状态对照表

| 业务操作 | 对应系统功能页 | 常见系统内状态 | 说明 |
| :--- | :--- | :--- | :--- |
| **收票** | `收票登记` | 已收票 / 可流通 | 票据进入本企业票据池，可用于后续流转 |
| **背书** | `票据背书` | 已背书 | 票据转让给前手/供应商，当前企业不再拥有所有权 |
| **质押** | `票据质押` | 已质押 / 占用中 | 票据作为担保物被锁定，无法进行其他流转操作 |
| **解质** | `质押解除` | 已收票 / 可流通 | 担保解除，票据恢复正常在库状态 |
| **贴现** | `票据贴现` | 已贴现 | 票据提前卖给银行，资金已折价到账 |
| **托收** | `票据托收` | 托收中 → 已结清 | 票据到期向承兑人要钱，资金全额到账后生命周期结束 |
