## 放款关联收款流水

```sql
-- 相关SQL
SELECT t.id, t.FMS_BUSINESS_NO, t.TRANSACTION_NO, t.TRANSACTION_SERIAL_NUMBER, t.BILL_RECEIPT_NO, t.BILL_RECEIPT_URL, t.BILL_RECEIPT_ID
FROM zfs_fms.fms_transaction_details_his t
WHERE FMS_BUSINESS_NO = 'd0f29f0e3ceb0f78f9c293f66a198bae37846184ff3ff7d51e982e902b81dde7';


SELECT t.id, t.RECEIPT_NO, t.BATCH_NUMBER, t.BATCH_NUMBER_SYS, t.TOUFO
FROM zfs_fms.fms_electronic_receipt t
WHERE RECEIPT_NO = 'NDEwMTE2NDUyOTQ='


select t.id as 'detail_id', t.TRANSACTION_NO,  t.FMS_BUSINESS_NO, t.BILL_RECEIPT_ID, t.BILL_RECEIPT_NO, t.BILL_RECEIPT_URL
, e.id as 'receipt_id',  e.RECEIPT_NO,  e.SYNC_FILE_URL, e.BATCH_NUMBER_SYS
from fms_transaction_details_his t
    left join fms_electronic_receipt e on e.RECEIPT_NO = t.BILL_RECEIPT_NO
where t.ENABLED_FLAG = '0'
 and (t.BILL_RECEIPT_NO is not null and t.BILL_RECEIPT_NO <> '')
and t.id = 'c50da509b9aff329fb5e3139322e629b'

```

### 交易流水
- fms_transaction_details_his
ID： 
FMS_BUSINESS_NO： 资金系统内生成的唯一KEY
TRANSACTION_SERIAL_NUMBER： 银行唯一标识
TRANSACTION_NO: 交易流水号

### 电子回单
- fms_electronic_receipt
ID: 
RECEIPT_NO: 回单编号
BATCH_NUMBER：交易流水号
BATCH_NUMBER_SYS： 系统交易流水号
TOUFO: 银行返回的电子回单唯一标识
